# 고등 연산 학습지 전수 점검

기준: 2022 개정 수학과 교육과정의 과목 구조와 현재 학습지의 실제 문항을 대조한다.

상태:

- `충분`: 핵심 계산 유형이 학습지에 구현됨
- `보강`: 학습지는 있으나 필수 유형이 일부 빠짐
- `누락`: 대응 학습지가 없음
- `재분류`: 교육과정 수준 또는 과목 배치를 바로잡아야 함

|---|---|---|

| 나머지정리·인수정리 | 충분 | 일차식·이차식으로 나눈 나머지까지 구현 |

| 이차함수와 이차방정식 | 누락 | 그래프와 근의 관계 학습지 추가 |

| 연립방정식·여러 부등식 | 보강 | 고차·분수·무리 유형은 있으나 교육과정 순서 재구성 |
| 순열과 조합 | 충분 | 기존 순열·조합 및 이항정리와 연결 |
| 행렬 | 충분 | 뜻, 성분, 덧셈·뺄셈·실수배·곱셈 구현 |

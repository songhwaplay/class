# V74 변경 사항

## 메시나 해협 양방향 통항 보정

- V73에서 남쪽에서 북쪽으로는 간신히 지나가지만 북쪽에서 남쪽으로 접근할 때 충돌 판정에 걸리던 문제를 수정했다.
- 메시나 해협의 이동 판정 폭을 한 단계 넓혀 대각선 진입과 클릭 이동에서도 양방향으로 통과할 수 있게 했다.
- 남→북·북→남을 각각 실제 서버 이동 간격과 충돌 후보 순서로 재현하는 회귀 테스트를 추가했다.
- 시칠리아 내륙, 칼라브리아 내륙, 팔레르모가 바다로 잘못 판정되지 않는 것도 함께 검사한다.
- 브라우저가 이전 지형 스크립트를 사용하지 않도록 캐시 버전을 V74로 올렸다.

# V75 변경 사항

## 구면 항해와 현실 속력

- 등거리 원통도법 지도에서 고위도 동서 이동을 `1 / cos(위도)`로 보정했다.
- 키보드 이동, 클릭 목적지, 해류 표류, 근접 판정, 이동 거리 기록을 같은 구면 기준으로 통일했다.
- 고위도에서 긴 한 틱 이동이 육지를 건너뛰지 않도록 작은 충돌 검사 단계로 나눴다.
- 극점의 수학적 특이점에서는 경도 배율을 최대 50배로 제한한다.

## 현실적인 시간과 속력 표시

- 기준 해상 이동속력은 약 8.8km/h(4.8노트), 육상 최대 이동속력은 약 6.4km/h로 환산된다.
- 바람·해류·지형·피로·보너스까지 적용된 실제 구면 이동거리를 게임 시간으로 나눠 HUD에 km/h로 표시한다.
- 화면 이동 템포를 유지하면서 현실 속력을 사용하기 위해 게임 시간을 초당 8시간에서 30시간으로 조정했다.
- 정지 상태는 `0 km/h`, 도시 내부는 `—`로 표시한다.
- 기본 이동은 현실적인 속도를 유지하되, 완주 보상은 UFO 콘셉트의 10·5·3배 초고속 이동을 그대로 유지한다.

## 검증

- 북위 80도에서 동서 지도 이동이 적도의 5.76배가 되면서 실제 km/h는 동일하게 유지되는 단위 테스트를 추가했다.
- 실제 서버 스냅샷에서 이동 중 속력과 정지 후 0km/h가 전달되는 통합 테스트를 추가했다.

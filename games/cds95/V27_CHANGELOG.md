# CDS95 v27 변경사항

## 원작 도시 전체 연결

- 원본 `CDS95.EXE`의 도시 테이블에서 도시 이름 225개와 지도 좌표를 추출했다.
- 도시를 원작 지도 좌표에 표시한다.
- 항구 도시는 바다 접근성에 따라 138곳으로 분류해 입항·출항·육상 탐험 전환을 지원한다.
- 항구가 아닌 도시 87곳도 지도에 이름과 위치가 표시되며 도착 미션 목적지로 선택할 수 있다.

## 모든 항구 도시의 이동 흐름

- 바다 → 항구 도시 입항
- 항구 도시 → 바다 출항
- 항구 도시 → 육상 탐험대 출발
- 육상 → 항구 도시 진입
- 도시별 입출항·육상 전환 시간은 기존 공용 시간 기준을 그대로 사용한다.

## 지도 이름 표시

- 항구 도시는 푸른색, 내륙 도시는 황금색으로 표시한다.
- 원작의 대도시 표시는 조금 크게 표시한다.
- 화면이 복잡한 지역에서는 겹치는 이름을 줄이고, 확대하면 더 많은 이름이 보인다.
- 현재 미션 목적지는 항상 우선 표시한다.

## 데이터 파일

- `data/catalog/original-cities.json`: 원작 도시 225개
- `data/catalog/places.json`: 반도·해협·폭포 등 지형 목적지

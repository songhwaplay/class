# CDS95 v26 변경사항

## 교사 출발 신호 방식

- 교사가 도착지 1곳과 출발 도시 4곳을 먼저 배포합니다.
- 학생은 출발 도시만 선택하고 해당 도시 앞바다에서 대기합니다.
- 학생이 도시를 골라도 키보드, 터치, 클릭 이동, 입출항, 상륙은 작동하지 않습니다.
- 접속한 학생 전원이 출발 도시를 선택해야 교사 화면의 `전원 출발` 버튼이 활성화됩니다.
- 한 명이라도 선택하지 않았으면 서버가 출발을 거부하고 학생 이름을 알려줍니다.
- 교사가 `전원 출발`을 누른 순간부터 전원이 동시에 움직일 수 있습니다.
- 출발 전에 물·식량, 피로도가 변하지 않습니다.
- 목적지 도착 후에는 다시 움직이지 못하며 순위가 고정됩니다.
- 미션이 없을 때도 학생은 임의로 움직일 수 없습니다.

## 교사 화면

1. `미션 준비 · 출발지 선택 시작`
2. 학생들의 출발 도시 선택 확인
3. `전원 출발`
4. TV 순위판 확인

## 학생 화면

- 미션 전: 교사의 미션 설정 대기
- 도시 선택 전: 출발 도시 4곳 중 선택
- 도시 선택 후: 교사 출발 신호 대기
- 교사 출발 후: 목적지까지 이동

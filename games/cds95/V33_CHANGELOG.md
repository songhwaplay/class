# v33 변경사항

- 교사 화면의 도시 목록을 원작 권역 및 HEX 순서대로 정렬했습니다.
- 도시 목록 순서: 이베리아 → 프랑스 → 네덜란드 → 이탈리아 → 브리튼 → 독일 → 동유럽 → 북유럽 → 발칸 → 북아프리카 → 서아프리카 → 동아프리카 → 흑해 → 근동 → 중동 → 중앙아시아 → 인도 → 동남아시아 → 동북아시아 → 서인도제도 → 중앙아메리카 → 남동아메리카 → 남서아메리카.
- 각 권역 내부 도시도 가나다순으로 다시 섞지 않고 원작 HEX 순서를 유지합니다.
- 출발 도시 4개 목록도 같은 원작 순서를 사용합니다.
- 주요 지형은 도시 목록과 분리해 별도 그룹으로 표시합니다.
- 기존 검색 기능은 유지합니다.

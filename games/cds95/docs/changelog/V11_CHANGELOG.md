# v11 변경사항

## 실제 지명 기반 미션 제작

- 교사가 좌표를 직접 입력하지 않고 실제 도시·지형을 목록에서 선택합니다.
- 지명에는 한글명, 영문명, 대륙, 지역, 위도·경도, 사회과부도 단서가 포함됩니다.
- 위도·경도는 원작 세계지도 좌표로 변환되며 바다·육지 타일에 맞게 자동 보정됩니다.

## 단계형 미션 엔진

- 물품 구입
- 배에 적재
- 해상 이동
- 항구 상륙
- 육상 탐험
- 지형 조사 또는 물품 전달
- 최종 완료

현재 단계와 적재 물품만 저장하며, 이동 경로나 장문의 탐험 기록은 남기지 않습니다.

## 기본 템플릿

1. 도시 간 물자 운송
2. 주요 지형 탐험
3. 탐사대 보급·지형 조사
4. 주요 해협·곶 통과 항로

## 기본 미션 예시

- 리스본의 코르크를 탕헤르로
- 나이아가라 폭포 측량 장비 전달
- 미시시피강 하구 탐사대 보급
- 크림반도 지도 제작
- 나일강 삼각주 조사
- 지브롤터 해협 통과 항로
- 희망봉 항로 조사
- 사하라 사막 조사대에 식수 전달
- 파나마 지협 조사
- 안데스산맥 표본 수집대 보급

## 사회과부도 활용

학생 화면에 실제 지형 이름과 위치 단서를 명확히 제시합니다. 기본값은 게임 지도에 정답 위치를 표시하지 않는 방식이며, 교사는 필요에 따라 넓은 범위 또는 정확한 지점 표시를 선택할 수 있습니다.

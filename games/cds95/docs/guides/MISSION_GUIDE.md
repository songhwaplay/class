# 교사용 미션 제작 안내

## 가장 간단한 방법

교사 화면에서 `기본 미션팩`을 고르고 `반 전체에 배포`를 누릅니다.

예시:

- 나이아가라 폭포 측량 장비 전달
- 크림반도 지도 제작
- 지브롤터 해협 통과 항로
- 사하라 사막 조사대에 식수 전달

## 직접 조합하기

### 도시 간 물자 운송

1. 출발 항구 도시 선택
2. 목적 항구 도시 선택
3. 물품 선택
4. 구입 장소와 전달 장소 입력
5. 지도 표시 방식 선택

학생 미션 예시:

> 사회과부도에서 리스본과 탕헤르의 위치를 각각 찾고, 두 도시가 접한 바다와 이동 방향을 확인하세요. 리스본 시장에서 코르크를 구입한 뒤 탕헤르 항구 창고에 전달하세요.

### 주요 지형 탐험

1. `주요 지형 탐험` 선택
2. 실제 지형 이름 선택
3. 지도 표시 방식 선택

학생 미션 예시:

> 사회과부도에서 나이아가라 폭포를 찾아 표시하고, 이리호와 온타리오호 사이에 있다는 점을 확인하세요. 게임에서 가까운 항구에 상륙한 뒤 폭포 조사 지점까지 이동하세요.

### 탐사대 보급·지형 조사

1. 출발 항구 도시 선택
2. 목적 지형 선택
3. 운송 물품 선택
4. 조사대나 연구소 등 전달 장소 입력

게임이 자동으로 다음 단계를 만듭니다.

```text
출발 항구에서 물품 구입
→ 목적지 근처 항구로 항해
→ 탐험대 상륙
→ 실제 지형까지 육상 이동
→ 조사대에 물품 전달
```

### 주요 해협·곶 통과

출발 도시, 통과할 해협·곶, 목적 도시를 선택합니다.

```text
출발 항구에서 항로 시작
→ 실제 해협·곶 통과
→ 목적 항구 도착
```

## 지도 표시 방식

- `표시 안 함`: 사회과부도에서 직접 찾기. 기본 권장값
- `넓은 범위`: 대략적인 탐색 구역만 표시
- `정확한 지점`: 연습이나 지원이 필요한 학생에게 사용

학생 화면의 집 모양 버튼도 `정확한 지점`일 때만 자동 이동을 지원합니다. 표시 안 함과 넓은 범위에서는 사회과부도를 보도록 안내합니다.

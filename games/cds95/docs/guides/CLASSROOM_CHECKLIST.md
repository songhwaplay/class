# 수업 전 점검표

- 학생에게 실명 대신 수업용 별칭 또는 번호를 사용하게 합니다.
- 반 코드는 수업마다 바꾸고 다른 반과 공유하지 않습니다.
- 교사 PIN은 GitHub 파일이 아니라 호스팅 환경변수에만 저장합니다.
- 학생 화면 `/`과 교사 화면 `/teacher.html`이 모두 열리는지 확인합니다.
- 실제 수업 인원에 가까운 크롬북으로 동시접속을 시험합니다.
- 가까운 학생끼리 상대 배·탐험대와 이름이 나타나는지 확인합니다.
- 수행 미션을 시험 배포하고 완료 여부가 교사 화면에 반영되는지 확인합니다.
- 전체 일시정지 중 이동과 날짜가 모두 멈추는지 확인합니다.
- 추가 입장 잠금과 학생·반 전체 리스본 복귀를 시험합니다.
- 학생 화면에 탐험일지나 상세 이동 기록이 표시되지 않는지 확인합니다.
- 교사 화면에는 현재 위치·상태·완료 여부만 표시되는지 확인합니다.
- 서버 재시작 뒤 현재 미션과 완료 상태가 필요한 범위에서 복원되는지 확인합니다.
- 수업 전날과 수업 직전에 `/health`가 정상인지 확인합니다.

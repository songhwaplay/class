# v23 변경사항

- 교사 화면을 도착지 1곳, 출발 도시 4곳, 실시간 순위판만 남기도록 단순화했습니다.
- 학생 화면에서 미션과 출발 도시 네 곳이 자동으로 크게 표시됩니다. 출발 도시 선택 전에는 창을 닫을 수 없습니다.
- 선택 후에도 상단에 `출발 도시 → 도착지`가 계속 표시됩니다.
- 리스본 전원 복귀 및 학생별 복귀 기능을 제거했습니다.
- 비와 눈 애니메이션을 완전히 제거했습니다.
- 해류의 순방향 도움은 최대 30%로 높이고, 역방향 손해는 최대 8%로 줄였습니다. 횡류 효과는 14%입니다.
- 교사 접속 시 과거의 일시정지·입장 잠금 상태를 자동 해제합니다.

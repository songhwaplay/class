# CDS95 V41 변경사항

## 현실형 항해 영향 비율

- 바람은 범선의 주동력으로 설정
  - 완전한 순풍: 최대 +100%
  - 완전한 역풍: 최대 -65%
  - 지역별 실제 바람 세기에 비례해 상한보다 낮게 적용
- 해류는 보조 동력으로 설정
  - 일반 주요 해류의 완전한 순류: 최대 +40%
  - 멕시코만류·쿠로시오의 강한 핵심부: 최대 +80%
  - 완전한 역류: 최대 -23%
  - 횡류: 흐름 방향으로 함대를 밀어냄
- 서버 권위 이동 계산과 학생 화면 로컬 예측이 같은 수치와 같은 해류 데이터를 사용

## 바람 애니메이션

- 원작 `CLOUD.CDS`에서 추출한 구름 스프라이트 사용
- 큰 구름 6프레임과 작은 구름 6프레임을 독립적으로 재생
- 큰 구름이 갑자기 작은 구름으로 바뀌던 이전 12프레임 연속 재생 문제 수정
- 구름이 현재 바람의 방향과 세기에 맞춰 이동
- 이동 구간 끝에서는 서서히 사라지고 다시 나타나 순간이동이 덜 보이도록 처리

## 해류 애니메이션

- 해류 경로를 따라 이동하는 8프레임 물결 애니메이션 추가
- 물결 위치뿐 아니라 물결 모양 자체도 계속 변화
- 해류는 바다에만 표시되며 육지에서는 마스킹
- 바람의 구름과 해류의 물결을 시각적으로 분리
- 멕시코만류·쿠로시오의 강한 핵심부는 HUD에 `핵심부`로 표시

## 검증

- 지도·카탈로그·HTML 스크립트 문법 검사 통과
- 주요 해류 수치 테스트 통과
- 주요 바람 및 계절풍 방향 테스트 통과
- 해류 8프레임 애니메이션 테스트 통과
- 원작 구름 대·소 6프레임 분리 애니메이션 테스트 통과

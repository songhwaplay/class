# CDS95 v23

실시간 지리 도착 미션 플랫폼입니다.

## 수업 흐름

1. 교사가 도착 도시 또는 지형 한 곳을 고릅니다.
2. 교사가 출발 항구 도시 네 곳을 고릅니다.
3. 학생은 화면에 나타난 네 도시 중 한 곳을 선택합니다.
4. 학생이 목적지에 들어가면 자동 성공 처리됩니다.
5. 교사 화면에 도착 순위가 실시간으로 표시됩니다.

## 접속

- 학생: `http://localhost:3000/`
- 교사: `http://localhost:3000/teacher.html`
- 기본 교사 PIN: `2468`

## 실행

```cmd
npm install
npm start
```

비·눈 효과는 사용하지 않습니다. 해류는 화면의 흐름 애니메이션과 실제 항해 속도에 반영됩니다.

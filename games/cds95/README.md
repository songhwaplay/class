# CDS95 v54

실시간 교실용 세계 항해·육상 탐험 학습 플랫폼이다.

## 실행

```bash
npm start
```

- 학생: `http://localhost:3000/`
- 교사: `http://localhost:3000/teacher.html`
- 기본 교사 PIN: `2468`

## 현재 수업 흐름

1. 교사가 도착 도시 또는 지형을 하나 선택한다.
2. 교사가 출발 항구 도시 네 곳을 선택한다.
3. 학생은 출발 도시 하나를 고른다.
4. 교사가 `전원 출발`을 눌러야 모두 동시에 움직일 수 있다.
5. 목적지에 먼저 도착한 순서대로 교사 화면에 순위가 표시된다.
6. 도착한 학생은 메달을 달고 1위 10배, 2위 5배, 3위 3배 속도로 계속 자유 탐험한다.
7. 교사 세계지도에서 모든 학생의 실시간 위치와 목적지 근접 상태를 확인한다.


## 출발 도시 선택 반응

- 버튼을 누르는 즉시 선택한 카드가 강조된다.
- 선택 요청을 지연시키던 이중 애니메이션 프레임 대기를 제거했다.
- 선택창이 열려 있는 동안 구름·해류 애니메이션을 잠시 멈추고 배경 렌더링을 낮춰 클릭 입력을 우선 처리한다.
- 항해가 시작되면 구름과 해류 애니메이션은 정상적으로 다시 실행된다.

## 육상 지형 이동

- 평원·해안·강·숲·사막·산악·고산을 모두 통과할 수 있다.
- 지형별로 이동속도만 달라지며 고산은 기본 육상속도의 18%로 가장 느리다.
- 고산을 거대한 통행금지 구역으로 처리하던 이전 판정을 제거했다.
- 육상 탐험대가 바다로 직접 들어가거나 함대가 육지로 올라가는 이동수단 경계만 차단한다.


## 원작 세계지도 표시

- 웹에서 임의로 숲·사막·강 색을 덧씌우던 팔레트 마스크를 제거했다.
- 원작 `WORLD.CDS`의 타일 번호와 원작 `OCEAN.CDS`에서 추출한 타일 아틀라스를 그대로 조합한다.
- 중국·나일강 등에서 보이던 직선·사각형 색상 경계가 사라지고 원작 지형의 자연스러운 연결이 유지된다.
- 숲·사막·강의 이동속도 판정은 유지하지만 지도 그림 자체는 더 이상 임의 재색칠하지 않는다.

## 주요 해류 학습

- 세계 지리·지구과학에서 배우는 핵심 표층 해류 15종을 18개 해역 경로로 단순화해 표시한다.
- 바다 위의 8프레임 물결 애니메이션이 해류 경로를 따라 흐른다. 물결 자체도 움직이며 육지 위에는 표시되지 않는다.
- 현재 해류명, 난류·한류 구분, 흐름 방향, 순류·역류 효과를 학생 화면 상단에서 확인할 수 있다.
- 일반 주요 해류의 완전한 순류 효과는 최대 40%이며, 멕시코만류·쿠로시오의 강한 핵심부만 최대 80%까지 빨라진다. 완전한 역류는 최대 23% 느려진다.
- 횡류는 함대를 흐름 방향으로 옆으로 밀어 항로 선택에 영향을 준다.


## 주요 바람 학습

- 지구과학에서 배우는 북동·남동 무역풍, 남·북반구 편서풍, 극지방 동풍을 위도대별로 구현했다.
- 인도양·남아시아에서는 계절에 따라 여름 남서 계절풍과 겨울 북동 계절풍이 바뀐다.
- 원작 `CLOUD.CDS`에서 추출한 큰 구름 6프레임과 작은 구름 6프레임이 각각 실제 바람 방향으로 이동하며 자연스럽게 형태가 변한다. V54에서는 물리 수치와 별개로 화면 이동을 더 빠르게 해 흐름 방향이 즉시 보인다.
- 구름은 바람을 나타내며, 해류는 바다 위 물결선으로 따로 표시해 학생이 둘을 혼동하지 않도록 했다.
- 범선의 주동력인 바람은 완전한 순풍에서 최대 100% 가속하고, 완전한 역풍에서 최대 65% 감속한다. 실제 지역 바람 세기에 따라 효과는 이 상한보다 낮아진다.
- 학생 화면 상단에서 현재 바람 이름, 방향, 순풍·역풍 효과를 확인할 수 있다.


## 도시 내부와 도서관

- 원작 `CITYCG.CDS`에서 225개 도시 내부 화면을 추출해 각 도시와 정확히 연결했다.
- 육상 탐험대가 실제 도시 입구에 닿으면 `도시 들어가기` 명령이 나타난다. 항구 도시에서는 `도시 들어가기`와 `승선`을 따로 선택할 수 있다.
- 도시 입장과 퇴장은 즉시 처리되며 별도의 10일 소요나 전환 대기시간이 없다.
- 학생이 도시 내부나 도서관에 있어도 교사 화면 기준의 학급 날짜와 시간은 계속 흐른다.
- 모든 도시에서 도서관을 열 수 있으며, 공통 항해·지리 자료와 도시 권역에 맞는 지역 자료를 선택해 읽을 수 있다.
- 도시 화면에서는 지도 렌더링을 멈춰 저사양 기기의 불필요한 부하를 줄였다.

## 항구 접근과 게임 시간

- 상륙·승선 명령은 도시 근처가 아니라 실제 항구 출입 셀에 닿았을 때만 표시된다.
- 파나마는 태평양 항구 쪽에서만 상륙할 수 있으며 대서양 연안에서는 상륙 명령이 나타나지 않는다.
- 게임 시간은 교사가 전원 출발을 누른 뒤 초당 8시간씩 진행된다. 이전 버전보다 2배 빠르다.
- 도착 보상은 1위 10배, 2위 5배, 3위 3배이며 4위부터는 추가 배속이 없다.

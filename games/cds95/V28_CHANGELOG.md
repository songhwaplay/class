# CDS95 v28 변경사항

## 원작 도시 접근 규칙 복원

- v27의 해안 거리 추정 분류를 폐기했다.
- `CDS95.EXE`의 도시 225개 좌표와 `WORLD.CDS`의 실제 도시 마커를 직접 연결했다.
- 배로 들어갈 수 있는지는 도시 마커 바로 옆에 원작의 항해 가능한 바다 칸이 존재하는지로 판정한다.
- 육상 진입은 도시 마커 바로 옆의 원작 육지 칸에서만 가능하다.
- 결과: 바다 입항 가능 도시 136곳, 육상 전용 도시 89곳.
- 한양은 원작 지도에서 바다 진입 칸이 존재하므로 입항·출항 가능하다.
- 북경·카이로·포토시는 원작 지도에서 바다 진입 칸이 없으므로 육상으로만 접근한다.

## 미션 성공 판정

- 도시 도착 미션은 도시 주변에 접근하는 것만으로 끝나지 않는다.
- 실제로 해당 도시의 입항 또는 육상 진입 절차를 마쳐 `city` 상태가 된 뒤 성공한다.

## 검증

- 225개 도시 모두 원작 도시 마커 존재 확인
- 모든 바다 진입 칸이 실제 바다 타일인지 확인
- 모든 육상 진입 칸이 실제 육지 타일인지 확인
- 한양·리스본·런던 바다 입항 확인
- 북경·카이로·포토시 육상 전용 확인
- 학생 33명 동시 출발 선택 및 교사 출발 신호 테스트 통과
